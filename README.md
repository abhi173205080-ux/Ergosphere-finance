# ⚡ Ergosphere – Your Financial Command Center

**Track expenses, set budgets, visualise spending, and stay on the right financial path – all in one beautiful dashboard.**

Built with ❤️ for my father.

---

## 🌟 Features

- 📋 **Expense Ledger** – Full spreadsheet‑like editor (add, edit, delete with auto‑saved categories).
- 💰 **Income Tracking** – Log multiple income sources and see net cash flow.
- 📊 **Interactive Dashboard** – Bar, pie, and forecast charts that reveal spending patterns.
- 🎯 **Envelope Budgeting** – Set monthly limits per category with progress bars and overspend alerts.
- 🔄 **Recurring Expenses** – Auto‑add fixed costs (rent, subscriptions) and get bill reminders.
- 💵 **Net Worth & Investments** – Track assets, liabilities, and portfolios (stocks, mutual funds, crypto).
- 🎯 **Savings Goals** – Define targets and monitor progress visually.
- 📁 **Import / Export** – Bulk‑upload via CSV or download your ledger.
- 🌙 **Dark / Light Theme** – Toggle for comfort.

## 🚀 Quick Start (Local Setup)

### 1. Clone the repository
```bash
git clone https://github.com/abhi173205080-ux/Ergosphere-finance.git
cd Ergosphere-finance
```
### 2. Install dependencies
```bash
pip install -r requirements.txt
```
### 4. Run the app
```bash
streamlit run app.py
```

The app will open at http://localhost:8501.
